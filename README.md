# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript. It fetches real-time weather data using an API and displays the temperature based on user input.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Displays real-time temperature  
- ⚡ Fast and lightweight  
- 🎨 Clean and simple UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Fetch API)  
- Weather API

## ⚙️ How It Works

1. User enters a city name  
2. App sends request to weather API  
3. API returns weather data  
4. Temperature is displayed on screen
